## Using LaTeX on Linux (Ubuntu)

To compile and edit LaTeX documents on Ubuntu, it is recommended to install the following components:

### 1. Install LaTeX

Open a terminal and run:

```bash
sudo apt install texlive-full
```

### 2. Install VS Code and LaTeX Workshop extension

1. Download and install Visual Studio Code if you donâ€™t have it already.

2. Open VS Code and go to the Extensions tab (Ctrl+Shift+X).

3. Search for LaTeX Workshop and install it.

### 3. Using LaTeX in VS Code

* Open .tex file in VS Code.

* Use the Build LaTeX Project button (or Ctrl+Alt+B) to compile to PDF.

* You can view the compiled PDF inside VS Code using the built-in preview.
